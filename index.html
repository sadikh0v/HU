HU Reacts sisteminin quruluşu

Admin panel

Sorğu formaların idarə olunması: yaradılması, düzəliş olunması, unikal link-lərin generasiyası, QR kodun yaranması, hesabatlar və infoqrafikaların formalaşması məhz admin paneldə həyata keçirilir

Sorğu forması

Respondent tərəfindən doldurulan forma. Suallara cavab növləri:
• emoji, stars, single select,
• multiselect, user input

API

HU Reacts xidməti müştərinin daxili sistemi ilə inteqrasiya olunması üçün interfeys təqdim edir.
• Personal linklərin yaradılması
• Feedback-lərin sorğulanması
• Hesabatların yüklənməsi

Üstünlüklər

HU Reacts sorğu formaları...

Sizin korporativ üslubunuza uyğunlaşır

Standart formaların mövcud olmasından əlavə rəng, üslüb və dizayn sizin korporativ qaydalara uyğun olaraq formalaşa bilər. Sualların yaradılması, onun mütləq olub olmaması, görünüşün idarə edilməsində sizə kömək olacaq düstürlar mövcuddur.

İki fərqli dildə tərtib oluna bilir

Böyük müştəri kütləsinə uyğunlaşmaq və bütün seqmentlərə əlçatan olmaq üçün sorğu formalarını iki dildə, yəni Azərbaycan və İngiliscə tərtib etmək olar.

Sizi xəbərdar edir

Müştəri xidməti mənfi dəyərləndirsə və ya şərh edərsə HU Reacts sorğu formaları üzrə ммасul шгхслəрин elektron ünvanına dərhal млзьтат ötürür.

Məmnuniyyət indeksi formalaşdırır

Sorğuların cavablarının sistem tərəfindən avтоматик апализи нəticəсиндə мммнуниyyəт индекси формалашдыр. Beləliklə siz həм мусььт, həм мэнфи рэй вермис мусььтылэрин саынь билиб, хидмэт кэйфиьэтинин артырэлмасы үчүн лазымлы аддымларын атылмасын тэмин эдэцэксиниз.

Sorğuları yaradın

Xidmət keyfiyyətinin artırılması məqsədilə müştəri məmnuniyyətinin ölçülməsi və ya hər-hansı sorğuların keçirilməsi üçün formaların yaradılması və kateqoriyalara bölünməsi imkanı admin paneldə yaradılmışdır

Linki göndərin

Sorğuların aparılması üçün unikal linklərin generasiyası mümkündür.
Bu funksional sorğu formalarını yüngülləşdirir və statistikanı daha dəqiq aparmağa imkan verir.

Hesabat əldə edin

Sorğunun analitik hesabatının aparılması məqsədilə cavabların Excel formatına export olunması və ya infoqrafika şəklində statistikanın əldə olunması imkanı mövcuddur.

Tez-tez verilən suallar

HU Reacts nədir?

HU Reacts müştəri məmnuniyyətinin ölçülməsi və ya sorğurların keçirilməsin məqsədilə formaların yaradılması və sorğu formalarına uyğun doldurulmuş cavabların detallı analizi, statistikasına nəzarət etmək imkanı verən platformadır.

Sorğunun linki bütün cihazlarda açılır?

Bəli. HU Reacts sorğusu üçün generasiya olunmuş link həm android, həm ios, həm də web mühitdə rahatlıqla açılıq.

Bir şəxs bir sorğuya bir neçə dəfə cavab versə, statistikam yalnış olacaq?

1 şəxs 1 sorğuya 1 dəfə cavab verə bildiyinə görə, hesabat və statistikanız yalnış olmayacaqdır.
